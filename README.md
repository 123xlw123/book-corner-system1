# “图书角”管理系统建模与版本控制实践

## 项目背景
为方便班级同学借阅图书，本项目计划开发一款简易Web应用“图书角”，核心功能包括图书信息录入、图书搜索、用户借阅和归还登记。

## 项目内容
本仓库包含“图书角”系统的UML建模文件及版本控制记录，具体如下：
- **用例图**：展示“用户”和“管理员”的核心功能。
- **类图**：包含“图书”“用户”“借阅记录”三个核心类及它们之间的关系。

## 版本控制说明
本项目通过Git进行版本管理，关键提交历史如下：
1. 项目初始化：提交README.md说明文档。
2. 提交用例图：上传系统用例图文件。
3. 提交类图：上传系统类图文件。
